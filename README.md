# 🧠 Zenith – AI Mental Wellness Companion



Zenith is a **therapy-style AI mental wellness web application** designed to simulate human-like emotional support conversations. It acts as a virtual companion that listens, responds empathetically, and helps users reflect on their thoughts and emotions.



This project is built entirely without external AI APIs, making it lightweight, privacy-friendly, and fully offline-capable.



---



## 🌟 Features



- 💬 Human-like conversational therapy experience  

- 🧠 Emotion-aware response engine (rule + context-based NLP)  

- 📊 Session-based mood tracking system  

- 📝 Psychological session summary generation  

- 🌙 Calm, dark-themed therapy UI (Calm/Headspace inspired)  

- ⚡ Fully offline (no API dependency)  

- 🧩 Lightweight Flask backend  



---



## 🏗️ Tech Stack



- **Frontend:** HTML, CSS, JavaScript, Bootstrap  

- **Backend:** Flask (Python)  

- **Logic:** Rule-based NLP + session memory system  



---



## 🧠 How It Works



Zenith does not use external APIs. Instead, it simulates AI behavior using:



- Keyword + intent detection  

- Conversation memory tracking  

- Emotional state analysis  

- Context-based response generation  

- Randomized human-like response variation  



This allows Zenith to mimic a supportive therapist-style interaction.
